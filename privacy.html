Politica sulla Privacy

Ultimo aggiornamento: 1 novembre 2025

Questo sito web è un portfolio personale gestito da MatteDev.

1. Dati Personali

Questo sito web non raccoglie, memorizza o elabora alcun dato personale dai visitatori. Non utilizza moduli di contatto, sistemi di login, strumenti di tracciamento o cookie di profilazione.

2. Servizi Esterni

Alcuni servizi di terze parti (ad esempio, hosting, CDN, font o video incorporati) possono raccogliere dati tecnici come indirizzi IP o user agent, secondo le proprie politiche sulla privacy.

  • Hosting o dominio (GitHub Pages)
  • www.w3.org

3. Cookie

Questo sito web non utilizza i propri cookie. Eventuali cookie tecnici da fornitori di terze parti sono gestiti direttamente da tali fornitori.

4. Diritti dell'Utente

Poiché non vengono elaborati dati personali, non ci sono richieste di accesso, modifica o cancellazione. Per qualsiasi domanda, contattare: support@mattedev.com.
